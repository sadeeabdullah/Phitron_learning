# Sum of Two Numbers

**Time Limit:** 1 second  
**Memory Limit:** 2.93 MB  

## Statement
Take two integers **A** and **B** as input and output their summation.

## Constraints
-10^9 <= A,B <= 10^9

## Input format
- You will be given **A** and **B** separated by a space.

## Output Format
Output their summation

## Sample Input 1
5 7

## Sample Output 1
12
